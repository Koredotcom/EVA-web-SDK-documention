# Getting Started

AI4W Web SDK is a powerful JavaScript library that enables developers to integrate AI-powered conversational capabilities into their web applications. It provides components and utilities to create chat interfaces, manage conversation history, handle file attachments, and interact with AI agents across React, Angular, and Vue frameworks.

## Installation

AI4W Web SDK can be installed in **React 18.3.1+**, **Angular 12+**, or **Vue 3+** applications.

### Using npm

```bash
npm install eva-web-sdk
```

### Using yarn

```bash
yarn add eva-web-sdk
```

## Prerequisites

Before you can use AI4W Web SDK, you'll need the following credentials from your AI4W administrator:

1. **Access Token**: Authentication token for API access
2. **API URL**: Base URL for AI4W API endpoints
3. **Presence URL**: WebSocket server URL for real-time communication
4. **User ID**: Unique identifier for the user

## Next Steps

Once you have installed the SDK and obtained your credentials, proceed to the **[Initialization Tutorial](./tutorials/initialization)** to learn how to configure and initialize the SDK in your application.

## Browser Support

AI4W Web SDK supports all modern browsers that support:
- ES6+ JavaScript features
- WebSocket API
- Modern framework versions (React 18+, Angular 12+, Vue 3+)

Recommended browsers:
- Chrome/Edge 90+
- Firefox 88+
- Safari 14+
